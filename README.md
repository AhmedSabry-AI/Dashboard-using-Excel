Interactive Dashboard in Excel 📊

في المشروع ده اشتغلت على تحويل بيانات المبيعات إلى Interactive Dashboard باستخدام Excel، وده كان الـ workflow اللي اتبعته:

1️⃣ Data Import & Preparation
أول خطوة كانت استيراد البيانات باستخدام Power Query، وبعدها بدأت مرحلة Data Processing اللي شملت:

إزالة Duplicate values

التعامل مع Missing values

التأكد من Data Types لكل عمود عشان تكون البيانات جاهزة للتحليل.

2️⃣ Data Analysis using Pivot Tables
بعد تجهيز البيانات، استخدمت Pivot Tables لاستخراج أهم الأرقام والإحصائيات، وكمان حساب مجموعة من Key Performance Indicators (KPIs) اللي بتوضح أداء المبيعات بشكل واضح.

3️⃣ Data Visualization (Charts)
في مرحلة بناء الداشبورد استخدمت أكثر من نوع من الـ charts، وكل واحد له هدف معين:

Bar Chart: مناسب لمقارنة الأداء بين الفئات المختلفة مثل المنتجات أو المناطق.

Column Chart: استخدمته لإظهار التغير في القيم عبر الوقت أو مقارنة القيم بشكل واضح.

Pie Chart: بيساعد في توضيح distribution أو النسب المئوية لكل فئة من إجمالي البيانات.

4️⃣ Dashboard Design
بعد اختيار الرسومات المناسبة، نظمت الـ charts داخل Dashboard Layout بحيث تكون القراءة سهلة وواضحة للمستخدم.

5️⃣ Interactivity
عشان أخلي الداشبورد أكثر تفاعلية:

أضفت Slicers لفلترة البيانات بسهولة.

وعملت Refresh Button لتحديث الداشبورد بسرعة بعد أي تعديل في البيانات.

الهدف من المشروع كان بناء Dynamic Dashboard تساعد في فهم البيانات واتخاذ قرارات مبنية على أرقام واضحة.
